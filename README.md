# Web-dev-project
Final Project 
#  LOSONGO & CO ADVOCATES

A modern, responsive law firm website developed as a web development project. The website showcases the firm's legal services, attorneys, representative cases, legal articles, and contact information while providing a clean and user-friendly experience.

---

##  Features

* Responsive Bootstrap 5 design
* Professional landing page
* About Us page
* Legal Services page
* Attorneys page
* Representative Cases page
* Blog 
* Image Gallery
* Frequently Asked Questions (FAQ)
* Contact page
* Consultation booking page
* Dark Mode toggle with theme persistence
* Bootstrap Icons throughout the website
* Mobile-friendly layout

---

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
* JavaScript
* Git & GitHub
